# 🎮 최신 게임 쿠폰 자동 추출 도구
* 반복적인 커뮤니티 웹 페이지 모니터링을 자동화한 데이터 수집 도구입니다.

# 🔍 주요 기능
보안 강화 : account.txt 방식을 폐지하고 .env 환경변수 관리 시스템을 도입하여 민감 정보를 보호합니다.
코드 모듈화 : 각 페이지(Path)별 기능을 분리하고, 팝업 닫기 등 중복 로직을 utils.py로 통합하여 유지보수성을 극대화했습니다.
예외 처리 : 빈 except 구문을 제거하고 상세 에러 로그 출력 및 유효성 검사 로직(길이 제한, 문자열 패턴)을 강화했습니다.
동적 로딩 대응: 무한 스크롤 및 자바스크립트 기반 동적 버튼 클릭 로직을 통해 데이터 수집 성공률을 높였습니다.

# 🛠 사용 기술
Language: Python 3.12
Libraries: 
    Selenium: 웹 자동화 및 동적 페이지 크롤링
    Pyperclip: 시스템 클립보드 제어를 통한 보안 로그인(캡차) 우회
    Python-dotenv: 환경변수(.env) 기반 보안 설정 관리
    Regex (re): 정규표현식을 활용한 영문/숫자 혼합 쿠폰 패턴 정밀 추출

# 📂 구성 파일
.
├── main.py            # [실행 파일] 전체 수집 프로세스 통합 제어 및 역순 처리 로직
├── path1.py           # [로그인] 캡차 우회 및 2단계 인증 대기 로직
├── path2.py           # [이동] 대상 라운지 진입 및 iframe/레이아웃 대응
├── path3.py           # [검색] 키워드 기반 동적 검색 수행
├── path4.py           # [수집] 게시글 리스트 분석 및 특정 작성자(GM) 필터링
├── path5.py           # [추출] 본문 내 영문/숫자 혼합 쿠폰 코드 및 기한 정밀 분석
├── path6.py           # [저장] 최신순 데이터 적층(LIFO) 및 중복 저장 방지
├── utils.py           # [공통] 팝업 배너 처리 등 반복 로직 통합 모듈
├── .env.example       # 환경변수 설정 가이드 (ID/PW 템플릿)
├── .gitignore         # 민감 정보(.env) 및 캐시 파일 업로드 차단 설정
├── requirements.txt   # 프로젝트 의존성 관리 파일
└── README.md          # 프로젝트 설명서

⚙️ 실행 방법 (Setup Guide)
1. 사전 준비
Python 3.12 이상 설치 및 Chrome 브라우저 최신화가 필요합니다.

2. 환경변수 설정
.env.example 파일을 복사하여 .env 파일을 생성합니다.(또는 .example 문구 삭제)
내부의 NAVER_ID, NAVER_PW 항목에 본인의 계정 정보를 입력합니다. (로그인 없이 사용 시 비워둡니다.)

3. 필수 패키지 설치
pip install -r requirements.txt

4. 프로그램 실행
python main.py
(!주의: 로그인 단계에서 클립보드 복사-붙여넣기 로직이 작동하므로, 브라우저가 뜨면 마우스나 키보드 조작을 잠시 멈춰주세요.)

📝 결과 확인
수집된 결과는 프로젝트 루트 폴더의 coupons.txt에 저장됩니다.
가장 최근에 발견된 쿠폰이 파일의 최상단에 위치하도록 설계되어 있습니다.
